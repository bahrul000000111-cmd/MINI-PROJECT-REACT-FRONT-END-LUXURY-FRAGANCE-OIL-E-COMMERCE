# MINI-PROJECT-REACT-FRONT-END-LUXURY-FRAGANCE-OIL-E-COMMERCE
Project ini adalah sebuah Landing Page E-Commerce yang dibuat menggunakan React.js dan Tailwind CSS v4. Proyek ini merupakan tugas slicing UI untuk mengkloning tampilan website minyak wangi mewah (Frågra) dengan fokus pada detail desain yang bersih dan responsif.
